## 🚀 快速开始


### 开发

1. 修改代码
2. **更新文档** - 修改代码后必须更新 `HOME_PAGE_LAYOUT.md`
3. 运行部署脚本
4. 提交更改

**每次修改代码后，必须同步更新 `HOME_PAGE_LAYOUT.md` 文档！**

文档更新要求：
- 添加新组件 → 更新组件说明
- 修改功能 → 更新功能说明
- 修改样式 → 更新样式说明
- 修改交互 → 更新交互说明
- 添加文件 → 更新依赖关系
